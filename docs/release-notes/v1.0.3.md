# 연금나침반 v1.0.3

삼성생명 IRP(개인형 퇴직연금) 계좌의 펀드/원리금보장형 비중 조정을 AI(Claude·Gemini·GPT)의 도움으로 계획하기 위한 Windows 데스크톱 도구입니다.

> ⚠️ **본 앱은 삼성생명보험주식회사와 무관한 비공식(third-party) 도구이며, AI가 생성한 제안은 참고용 정보일 뿐 투자 권유가 아닙니다.** 모든 투자 결정과 그 결과에 대한 책임은 사용자 본인에게 있습니다.

## v1.0.3 변경 사항

- 보유 상품 중복 추가 차단: 자동완성 후보에서 이미 추가한 상품을 숨기고, 직접 입력으로 같은 이름을 등록하려 하면 안내 후 무시
- 보유 상품 행의 휴지통(삭제) 버튼 가시성 개선: 컬럼 헤더에 "삭제" 라벨과 툴팁 추가
- 휴지통 버튼 위치를 "적립금" 바로 뒤로 이동하여 필수 입력 직후 잘못 추가한 행을 곧바로 되돌릴 수 있도록 정리
- 보유 상품 목록 헤더와 행의 컬럼 정렬 어긋남 수정 (오른쪽으로 누적되어 밀리던 헤더가 행과 정확히 맞도록)

## 설치 방법

다음 두 파일을 함께 받으세요:

| 파일 | 용도 |
| --- | --- |
| `PensionCompass_1.0.3.0_x64.msixbundle` | 앱 본체 |
| `PensionCompass_1.0.3.0_x64.cer` | self-signed 코드 서명 인증서 |

### 1단계: 인증서 신뢰 등록 (PC당 최초 1회)

자체 서명 인증서이므로 받는 PC가 발급자(CN=Changhwan Bang)를 신뢰하도록 등록해야 합니다.

1. `PensionCompass_1.0.3.0_x64.cer` 더블클릭
2. **인증서 설치(I)** 버튼 클릭
3. 저장소 위치: **로컬 컴퓨터(L)** 선택 → 다음
4. 인증서 저장소: **모든 인증서를 다음 저장소에 저장(P)** → 찾아보기 → **신뢰할 수 있는 사람** 선택 → 확인 → 다음 → 마침

### 2단계: 앱 설치

`PensionCompass_1.0.3.0_x64.msixbundle` 더블클릭 → "설치" 버튼.

설치 완료 후 시작 메뉴에서 "연금나침반"으로 검색해 실행할 수 있습니다.

### 업데이트하는 경우

기존 버전이 이미 설치되어 있다면 새 .msixbundle을 더블클릭하면 in-place 업데이트됩니다 (인증서 재등록 불필요, 기존 입력 데이터 유지).

## 시스템 요구 사항

- Windows 10 17763(1809) 이상 / Windows 11
- x64 아키텍처
- AI 호출용 인터넷 연결 + 사용자 본인의 API Key (Anthropic Claude / Google Gemini / OpenAI 중 하나 이상)

## 데이터 보관

계좌 정보, 상품 카탈로그, API Key는 모두 사용자 PC의 앱 LocalState 폴더에만 저장되며 개발자에게 전송되지 않습니다. AI 호출 시에는 입력하신 보유 현황과 카탈로그가 사용자가 선택한 AI 공급자(Anthropic / Google / OpenAI)에 직접 전송됩니다.
