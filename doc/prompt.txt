첨부된 문서를 참고하여 다음과 같은 프로그램을 개발하려고 해요.

우선, 프로그램은 Visual Studio 2026으로 WinUI 빈 앱(패키지됨) 프로젝트를 선택하여 만들려고 해요.
언어는 C#, XAML을 사용해요. Windows 11에서 동작하는 앱이면 되고요, 패키지까지 되면 더 좋을 것 같아요.

프로젝트 이름은 SLifeIrpRebalancer가 좋을 것 같은데, 더 좋은 제안이 있으면 해주시면 감사하겠습니다.

이 앱의 기능은

1. 삼성생명 IRP에서 매수 가능한 상품 목록이 포함된 HTML 파일을 읽어들여서 펀드 목록과 펀드별 정보(예를 들어 3개월 수익률 등)를 
CSV 파일로 생성할 수 있어야 해요. 생성해야 할 CSV 파일은 2가지로, 하나는 원리금보장형 상품 목록, 다른 하나는 펀드 상품 목록이예요.
그리고, HTML파일로 부터 수익률의 기간을 판별할 수 있어야 해요. 수익률 기간은 다음과 같아요.
(1개월/3개월/6개월/1년/3년)
이것의 구현 자체는 어렵지 않을텐데, 이 수익률이 포트폴리오 구성 판단에 영향을 끼칠 수 있어서 CSV 파일에 반드시 포함이 되어야 해요.
즉, 컬럼에 '수익률(3년)' 이런 표현이 포함되어야 하겠죠.

그리고 상품을 운용하는 금융회사들도 CSV에 포함되면 좋을 것 같아요. 왜냐하면 연금 개시할 때 종신형 지급을 받고 싶으면
연금개시 직전에 삼성생명이 운용사인 상품들로만 운용되어야 한다고 고객센터에서 설명했기 때문이예요.
(아마도 다음 단계에서 이를 묻는 UI가 필요할 수도 있겠어요.)

2. 이제 1번에 생성된 CSV 파일을 먼저 입력받은 다음 (또는 HTML 파일을 입력받아 1번 과정을 거친 다음)
사용자로 부터 가입자의 현재 IRP 계좌의 상품 구성 정보를 입력받을 수 있어야 해요. 앞에서 입력받은 CSV 파일의 내용을 통하여
사용자가 편리하게 선택하여 입력할 수 있는 UI가 있어야 겠어요.

우선 총 적립금(현재 계좌 가치)을 먼저 필수로 입력받아야겠고요, 입금액과 운용수익은 선택사항으로 입력하면 좋겠어요.
그리고 보유상품별 적립금을 필수로 입력받아야 하고요, 수익률/연환산수익률/운용일수/좌수 입력은 항목마다 선택사항으로 하면 좋겠어요.

2-1. 앞의 내용을 입력받았으면, 실제로 매도(전량 또는 부분)할 대상 상품을 사용자가 선택할 수 있는 UI가 있어야겠어요.
또한, 만기가 있는 상품의 경우, 만기 예약용임을 사전에 물어보는 UI도 있어야겠어요. 왜냐하면 언제 실제 만기가 되는지는
중요하기 때문이지요. 아마도, 최대한 즉시 리밸런싱할 것인지, 또는 좀 더 있다가 할 것인지를 사전에 물어보는 UI가 공통적으로
필요하겠어요. 즉, 만기 예약인지 아니면 일반적인 리밸런싱인지를 물어봐야 하는 거죠.

3. 2번을 다 입력을 받았으면, 이제부터 중요한데, Gemini/GPT/Claude로 부터 API Key를 입력받고, 해당 AI에게 1,2번으로부터
입력받은 정보를 종합하여 쿼리를 수행하는 거예요. 쿼리를 어떻게 수행하는 게 좋을 지는 제가 해온 질의를 참고하세요.
(말 그래도 참고만 하세요.) 

4. 최종적으로 포트폴리오 제안을 받아서 pdf 등으로 출력하는 기능을 구현해주세요.

이상의 기능을 수행하는 앱을 만들어주세요.
참고로 doc 폴더에 SLifeIrpRebalancer_상세_설계서.md 를 올려놓았어요.
